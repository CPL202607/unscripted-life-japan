# 西南戰爭 3D 戰場全記錄

明治十年（1877）西南戰爭互動 3D 沙盤：從鹿兒島火藥庫事件、熊本城攻防、田原坂血戰，到城山最後決戰，十二幕重現日本最後的內戰。

純靜態單頁應用：Three.js（CDN）+ 原生 JavaScript，無建置步驟，開啟 `index.html` 即可執行（建議透過 HTTP 伺服器，例如 `python3 -m http.server`）。

- `index.html` — 頁面結構與樣式
- `data.js` — 十二幕史實資料（旁白、事件、兵力、運鏡、部隊路徑）
- `app.js` — 3D 引擎（程序化九州地形、部隊、特效、運鏡、播放器）

致敬靈感來源：[蔡依橙〈赤壁之戰 3D〉](https://iclab.ichentsai.tw/redcliffs-3d/)。

地圖為敘事示意，兵力數字為史家估計，各方記載不一。
